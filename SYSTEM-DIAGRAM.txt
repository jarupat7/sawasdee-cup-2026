═══════════════════════════════════════════════════════════════════════════════
DATA FLOW SEQUENCE
═══════════════════════════════════════════════════════════════════════════════

    STEP 1: PLAYER REGISTRATION
    ──────────────────────────
    Player fills form on Netlify
           ↓
    [Click Submit]
           ↓
    Browser sends HTTPS POST
           ↓

    STEP 2: BACKEND PROCESSING
    ──────────────────────────
    GAS receives form data
           ↓
    validation.gs: Validates input ✓
           ↓
    tagGenerator.gs: Generates Tag ID ✓
           ↓
    database.gs: Saves to Google Sheet ✓
           ↓
    emailService.gs: Sends confirmation email ✓
           ↓

    STEP 3: EMAIL CONFIRMATION
    ──────────────────────────
    Player receives email with:
    - Tag ID: SC2026-001-ABC123
    - Fee amount: 1,500 THB
    - Bank account details
    - Instructions: Use Tag ID as payment reference
           ↓

    STEP 4: PLAYER PAYMENT
    ─────────────────────
    Player goes to bank
           ↓
    Transfers money to: Sawasdee Cup 2026 account
           ↓
    Uses Tag ID as reference/note
           ↓
    Takes screenshot/receipt
           ↓

    STEP 5: PAYMENT EMAIL
    ───────────────────
    Player emails receipt to: finance.jadesport@gmail.com
           ↓
    Subject: [SC2026-001-ABC123] Payment Slip
    Attachment: Bank transfer screenshot
           ↓

    STEP 6: ADMIN VERIFICATION
    ──────────────────────────
    Admin receives email with receipt
           ↓
    Admin opens: Google Sheet → PAYMENT_TRACKING
           ↓
    Finds row with: SC2026-001-ABC123
           ↓
    Updates:
    - Payment_Received: Yes
    - Receipt_Date: 2026-01-16
    - Payment_Verified: Yes
    - Verified_By: Admin Name
           ↓
    GAS detects change (onEdit trigger)
           ↓

    STEP 7: AUTO CONFIRMATION EMAIL
    ──────────────────────────────
    System automatically sends:
    - To: player@email.com
    - Subject: Payment Confirmed!
    - Content: Registration confirmed, ready to compete
           ↓

    STEP 8: SUCCESS
    ──────────────
    ✓ Player is REGISTERED & PAID
    ✓ Status: CONFIRMED
    ✓ Ready to compete in tournament

KEY COMPONENTS & RESPONSIBILITIES
═══════════════════════════════════════════════════════════════════════════════

    🎨 FRONTEND (Netlify)
    ├─ User Interface
    ├─ Form Display & Interaction
    ├─ Input Validation (client-side)
    ├─ Error Message Display
    ├─ Loading indicators
    └─ Confirmation page display

    ⚙️ BACKEND (Google Apps Script)
    ├─ Receive form submissions
    ├─ Validate data (server-side)
    ├─ Generate unique Tag IDs
    ├─ Calculate registration fees
    ├─ Save registrations to database
    ├─ Log all activities
    ├─ Send confirmation emails
    ├─ Monitor payment status
    └─ Trigger payment confirmation emails

    💾 DATABASE (Google Sheets)
    ├─ Store registrations
    ├─ Store category definitions
    ├─ Log email send history
    ├─ Track payment verification
    └─ Provide admin access

    📧 EMAIL (Gmail)
    ├─ Send confirmation emails
    ├─ Send payment reminders (future)
    ├─ Send payment confirmation
    └─ Send tournament updates (future)

    🏦 MANUAL PAYMENT (Bank Transfer)
    ├─ Players transfer fee to bank account
    ├─ Include Tag ID as reference
    └─ Admin verifies receipt
